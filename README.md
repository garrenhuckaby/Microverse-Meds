# Med Scheduler

AI-enabled medication tracker with constraint-based scheduling.

## Project Structure
```
med_scheduler/
├── rules/          # Constraint definitions and medication rules
├── engine/         # Core scheduling engine
├── data/           # Drug database
├── tests/          # Test cases and canonical regimens
└── main.py         # Entry point
```

## Setup
```bash
# Clone the repository
git clone https://github.com/YOUR-USERNAME/med-scheduler.git
cd med-scheduler

# Install dependencies
pip install -r requirements.txt
```

## Usage
```bash
python main.py
```
